动态背景配置项

  • color ：线条颜色, 默认: ‘0,0,0’；三个数字分别为(R,G,B)
  • opacity: 线条透明度（0~1）, 默认: 0.5
  • count: 线条的总数量, 默认: 150
  • zIndex: 背景的z-index属性，css属性用于控制所在层的位置, 默认: -1

添加访问量

在主题的_config.yml文件里面

开启之后无法统计，原因是js文件地址错误，在/theme/next/layout/_third-party/analytics/busuanzi-counter.swig文件中修改地址即可。

在本地查看发现访问量异常地大，这是正常现象，因为使用不蒜子统计的用户都使用同一个存储空间，如果你的URL和别人重复，就会出现数据量异常。这样的情况一般出现在你使用localhost:4000访问自己在本地部署的网页的时候。

遇到不蒜子无法统计问题，原因是https://dn-lbstatics.qbox.me/busuanzi/2.3/busuanzi.pure.mini.js无法访问，在/theme/next/layout/_third-party/analytics/busuanzi-counter.swig文件中修改地址即可

GitHub hexo绑定自己的域名

1.获取IP地址，通过ping xxx.github.io得到。

2.打开阿里云域名配置页面，点击解析，设置一下A和CNAME

2.在Github的xxx.github.io项目,进入【Settings】标签页,在【Custom domain】功能中,填写自己的域名。

4.在hexo项目的source文件夹下，新建一个名为CNAME的文件，写入自己的域名即可。
