react配置多页面应用

发表于 2018-10-05 | 分类于 react

1.新建一个项目

2.暴露配置

可以看到多了一个config和一个script文件夹

3.在src下新建一个admin.js文件

4.修改config下webpack.config.dev.js文件中的entry

5.修改output，对应生成boundle.js

6.修改plugins中的HtmlWebpackPlugin

在页面上修改路径，发现不管是什么路径都是index.html的页面内容，因为在开发环境中如果想通过地址访问不同的页面，需要修改webpackDevServer.config.js中的historyApiFallback

在webpack.config.dev.js中引入glob工具

引入path对象主要用到两个方法

path.basename(p, [ext]) 特点：返回指定的文件名，返回结果可排除[ext]后缀字符串。示例：path.basename(‘/foo/bar/baz/asdf/quux.html’)=>’quux.html’

path.extname(p)特点：返回指定文件名的扩展名称 .示例：path.extname(‘index.html’)=>’.html’

执行npm run build 后打开index.html空白，查看控制台信息

在package.json中添加homepage属性

重新打包即可
